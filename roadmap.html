Roadmap

  • Project Hash Start

    July 2020

    We are all pixel game lovers, but the pixel assets we like belong to the game maker rather than us, and even the assets we create do not belong to us. So we decided to act. To make the pixel NFTs and assets belong to the players.

  • Beta game launch

    August 2021

    Some pixel game lovers were invited to have a small-scale trial and switch the project on Polygon chain.

  • HashLand Blind Boxes

    September 2021

    There are 2500 pieces of HashLand in the game, and the first batch of 1000 plots of land are open for blind box sales. The size of the plot ranges from 50 * 50 to 500 * 500, which are randomly obtained from the blind box.

  • Alpha game launch

    November 2021

    Plot owners can enter the game to start mining, explore on the official plot and play the game,find rare NFT assets. The plot construction editing tool will be launched in the next version.

  • Land editing tool

    January 2022

    The plot editing tool is open. All players can enter the game, explore freely, go to the land owner's plot to mine (the landlord draws a commission), and edit and sell their own NFT in the game. According to the vote, the NFT made by players can also have a mining bonus.

The game will come later!
